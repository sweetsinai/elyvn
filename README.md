## What Is ELYVN?

<table>
<tr>
<td width="60%">

**ELYVN is an autonomous AI receptionist and sales automation platform** built for local service businesses — dental clinics, HVAC, salons, spas, gyms, real estate, plumbing, and more.

When a customer calls, texts, or fills out a form — the AI handles everything:

- Answers the call in <1 ring
- Qualifies the lead with natural conversation
- Books the appointment on Cal.com
- Sends confirmation SMS
- Notifies the owner on Telegram
- Scores the lead 1-10
- Schedules intelligent follow-ups
- Tracks full revenue attribution

**No human intervention. No missed calls. No leads lost.**

## Features

<!-- VOICE CALLS -->
<details>
<summary><h3>📞 Voice Calls (Retell AI)</h3></summary>

| Capability | Details |
|:-----------|:--------|
| **AI Receptionist** | Answers inbound calls in <1 ring, 24/7/365. Natural conversation powered by Retell AI |
| **Real-Time Transcription** | Full call transcript captured and stored |
| **Sentiment Analysis** | Every call scored: positive / neutral / negative |
| **Quality Scoring** | Claude rates each call 1-10 with 2-line summary |
| **Auto Lead Creation** | New caller? Lead created automatically from phone number |
| **Human Transfer** | Press `*` or say "speak to someone" — instant transfer to owner |
| **Voicemail Detection** | Detects voicemail, schedules smart callback |
| **Outbound Calls** | AI can initiate calls for speed-to-lead callbacks |
| **Circuit Breaker** | 3 failures in 60s triggers 30s cooldown — prevents API hammering |

</details>

<!-- SMS -->
<details>
<summary><h3>💬 SMS Conversations (Telnyx + Twilio)</h3></summary>

| Capability | Details |
|:-----------|:--------|
| **Dual Provider** | Telnyx primary, Twilio fallback — auto-failover |
| **AI Auto-Reply** | Claude reads client knowledge base, generates contextual replies |
| **Confidence Scoring** | High / medium / low — low confidence escalates to owner |
| **Telegram Escalation** | Low-confidence replies forwarded to owner's Telegram for review |
| **TCPA Compliance** | STOP / UNSUBSCRIBE detection, opt-out tracking, rate limiting |
| **5-Min Rate Limit** | Prevents accidental spam — max 1 SMS per 5 min per phone |
| **Ed25519 Verification** | Telnyx webhook signatures cryptographically verified |
| **Deduplication** | 5-minute lookback window prevents duplicate sends |

</details>
